# Генератор відео
## Створи своє відеопривітання !


Обери фотографії за ключовими словами та створи відео на їх основі.


## Команди

Для запуску додатку:

### `npm start`

Для білду додатку:

### `npm run build`


## Для створення відео, необхідно запустити сервер

Для запуску серверу з кореневої директорії (обов'язково):

### `node server.js`
